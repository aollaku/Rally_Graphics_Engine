Two-step graphic trigger update
==============================

Graphic buttons now use a safe Preview-first workflow:

1. First press on any graphic/page/stage button sends that graphic to Preview only.
2. Press the same graphic/page/stage button again to send it to Program / Live Output.

This prevents accidental live output changes while still allowing fast operation during a live rally.

Scene Manager controls remain available for manual TAKE Preview -> Program, Clear Program, bugs/sponsor layer, clock layer and macros.
