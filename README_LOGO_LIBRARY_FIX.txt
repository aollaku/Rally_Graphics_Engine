Logo Library Fix
================

Changes:
- Bug/Sponsor and Logo switches now keep their state instead of bouncing back after a click.
- Added Logo Library in Graphics Settings.
- Upload PNG logos first, then select the active logo from the library.
- Transparent PNG files are preserved.
- Added Refresh Logos and No Logo buttons.
- Selected logo is applied live to Preview, Program and controller monitor.

Usage:
1. Open Controller > Designer / Graphics Settings > Bug / Sponsor / Clock.
2. Click Upload Logo First and choose a PNG.
3. Select the logo from Select Logo From Library.
4. Enable Bug / Sponsor Layer.
