Tablet controller update V3
- Added CLEAR PREVIEW button. This clears only the tablet/preview scene, not Programme/output.
- CLEAR PGM still clears only Programme/output graphics and leaves the preview monitor unchanged.
- Event name display now avoids generic "Results on the Web" titles and shows the useful rally/stage name, including names parsed from subtitles such as "... - LLANGOWER 1".
- ALL [AUTO] is now locked to the selected graphic and stage when started. It loops selected pages every 8 seconds until stopped.
- If the operator changes GFX, stage, or a manual page while ALL [AUTO] is running, ALL [AUTO] stops first so it cannot jump back to page 1 or air the wrong graphic.
