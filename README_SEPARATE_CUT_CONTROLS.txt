Separate CUT Controls
=====================

Added independent cut/off controls:

- CUT Bug removes only the Bug Text layer from Live Output.
- CUT Logo removes only the Logo layer from Live Output.
- CUT Clock removes only the Clock layer from Live Output.
- CUT Main Preview removes only the main selected graphic from Preview.
- CUT Main Output removes only the main selected graphic from Live Output.
- CUT Main Both removes only the main selected graphic from Preview and Live Output.

Overlay layers remain independent from main graphics. For example, cutting Stage Times will not remove Logo, Clock or Bug Text.
