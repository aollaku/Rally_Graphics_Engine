Clock / Bug / Sponsor graphics settings added.

Changes:
- Graphics Settings now includes a Clock / Bug tab.
- Clock can be enabled, resized, moved and opacity-adjusted from Graphics Settings.
- Bug / Sponsor can be enabled, resized, moved, opacity-adjusted and edited from Graphics Settings.
- PNG logo upload added for Bug / Sponsor layer.
- PNG transparency is preserved.
- Uploaded logos are stored in public/uploads/logos and exported as part of the app files.
- Logo width and opacity can be adjusted live with fine arrows or slider.
