Phase 8 - Broadcast Output Manager

Added a dedicated Broadcast Output Manager card under Outputs.

Includes profiles for:
- Program
- Preview
- NDI
- SRT
- YouTube
- Facebook
- Twitch
- Social Vertical

Each profile stores enable/disable state, label, resolution, aspect, transport type, URL/destination and operator notes.

Important: this feature does not move, resize or rewrite the graphics renderer. The stable 16:9 preview/output rendering remains unchanged.
