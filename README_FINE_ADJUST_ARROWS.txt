Fine adjustment arrows added to Graphics Designer.

Each graphics setting line now has left/right arrow buttons next to the slider/input.
The arrows change the value using the field step size:
- Scale: 0.01
- Position: 1px
- Width/Height: 10px
- Opacity/Look: 1%
- Blur/Radius: 1px
- Animation speed: 0.1x
- Animation duration: 10ms

The change is saved immediately and applied live to Preview and Program outputs.
